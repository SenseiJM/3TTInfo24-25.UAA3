Bonjour CSS !

CSS nous permet de styliser nos pages web. Il y a aussi une autre page associée à cet exemple.

Listes stylisées

On peut styliser des listes non ordonnées avec les puces suivantes :

  • disque
  • cercle
  • carré

Et on peut numéroter des listes ordonnées comme ceci :

  1. decimal
  2. lower-roman
  3. upper-roman
  4. lower-alpha
  5. upper-alpha
  6. et bien plus !
